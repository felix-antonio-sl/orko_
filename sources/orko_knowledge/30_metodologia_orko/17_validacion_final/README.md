# 17_validacion_final – VG4 e Informe de RELEASE

Esta carpeta contiene los artefactos de **validación final de la metodología ORKO** antes de declarar la RELEASE 1.0.0 (VG4, invariantes `I1`–`I8`).

- `01_validacion_trazabilidad_i1_i8.md` – análisis detallado por invariante, basado en `03_vg4_validation_map.md` y en los 6 casos de `40_implementacion_metodologia/ejemplos/`.
- `validation_final_report.md` – reporte ejecutivo de CAP-14–CAP-17/CAP-19:
  - Estado de cada CAP (kernel, casos, playbooks, VG4).
  - Auditorías globales (§2.1–§2.5: VOCAB, DEP_GRAPH, §0 FUNDAMENTO, templates/calculadoras, casos).
  - Síntesis por invariante I1–I8 (§3) con estado `PASSED`/`CONDITIONAL`.
  - Conclusión de RELEASE y backlog post-1.0.0 (§4–§5).

El Capitán usa este directorio, junto con `40_implementacion_metodologia/CHANGELOG.md`, como referencia principal para decidir la **RELEASE ORKO v1.0.0**.
