# Catálogo de Conocimiento: Orko Builder (Toolkit & Implementation)

Este catálogo indexa los recursos prácticos y herramientas para el Agente **Orko Builder** (Constructor).
Estos documentos proveen las calculadoras, plantillas, ejemplos y guías de uso del Toolkit v1.0.0.

## Fuentes Primarias (Nivel 4: Herramientas)

### 1. ORKO Toolkit v1.0.0
- **Directorio**: `ORKO_Toolkit_v1.0.0/`
- **Propósito**: Paquete de distribución para el usuario final.
- **Componentes**:
    - `00_COMIENCE_AQUI.md`: Guía de inicio rápido.
    - `01_DIAGNOSTICO/`: Herramientas para F1 (Context Assessment).
    - `02_EJECUCION/`: Plantillas para Fases de Desarrollo y Operación.
    - `03_REFERENCIA/`: Cheat sheets y glosarios.
    - `04_SOPORTE/`: Canales de ayuda.

### 2. Implementación Metodológica
- **Directorio**: `40_implementacion_metodologia/`
- **Propósito**: Recursos de soporte para la ejecución de la metodología.
- **Componentes**:
    - `calculadoras/`: Scripts y hojas de cálculo para métricas (H_org, V_org).
    - `ejemplos/`: Casos de uso de referencia y artefactos de muestra.
    - `dev_specs/`: Especificaciones para desarrolladores de herramientas Orko.
